# <a name="designing-your-microsoft-teams-app-with-basic-fluent-ui-components"></a>Entwerfen Ihrer Microsoft Teams-App mit grundlegenden Fluent-UI-Komponenten

Microsoft Teams-apps werden in der Regel mit den folgenden grundlegenden Fluent-UI-Komponenten erstellt. Diese Komponenten sind so flach wie möglich ausgelegt und können in verschiedenen Anwendungsfällen, Designs und Bildschirmgrößen verwendet werden.

Die Abbildungen auf dieser Seite zeigen, wie Komponenten in Teams standardmäßig (hell) und dunkle Designs aussehen.

## <a name="microsoft-teams-ui-kit"></a>Microsoft Teams-UI-Kit

Das Microsoft Teams UI Kit basiert auf <a href="https://fluentsite.z22.web.core.windows.net/" target="_blank">Fluent-Benutzeroberfläche</a>und enthält Komponenten und Muster, die speziell für das Erstellen von Teams-apps entwickelt wurden. Im UI Kit können Sie die hier aufgeführten Komponenten direkt in Ihrem Design erfassen und einfügen und weitere Beispiele für die Verwendung der einzelnen Komponenten anzeigen.
