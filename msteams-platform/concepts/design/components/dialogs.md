# <a name="dialogs"></a>Dialoge

Dialogfelder sollten sparsam verwendet werden, da sie störend sein können und für den Benutzer eine eindeutige Möglichkeit zum Beenden haben sollten.

---

## <a name="sizes"></a>Größen

Wir haben drei Dialog Größen zur Auswahl. Wählen Sie die Größe aus, die für die Inhalte geeignet ist, die Sie haben.
[!include[Dialog sizes](~/includes/design/dialogs-image-sizes.html)]

---

## <a name="buttons"></a>Schaltflächen

Schaltflächen sind rechtsbündig ausgerichtet.
Alle anderen Elemente (Kontrollkästchen, Hilfetext usw.) sollten linksbündig ausgerichtet sein.
Duplizieren Sie keine negativen Aktionen (geben Sie niemals gleichzeitig eine Schaltfläche "X" und "Abbrechen" an.).
"Apply"-Schaltflächen sollten mit "Cancel"-Schaltflächen gekoppelt werden.
[!include[Dialog buttons](~/includes/design/dialogs-image-buttons.html)]

---

## <a name="scrolling"></a>Scrollen

In einigen Fällen wird der Bildlauf durch den Inhalt durchgeführt. Der Titel des Dialogfelds und andere wichtige Informationen bleiben bestehen.
[!include[Dialog scrolling](~/includes/design/dialogs-image-scrolling.html)]

---

## <a name="padding"></a>Padding

Der Abstand um alle vier Seiten des Dialogs sollte Bund sein.
[!include[Dialog padding](~/includes/design/dialogs-image-padding.html)]

---

## <a name="typography"></a>Typografie

Wir verwenden Segoe UI Semibold unter 18pt (Titel2) und $App-Black für Dialogtitel. Für den Textkörper verwenden wir Segoe UI Regular bei 14PT (Base) und $App-Black.
[!include[Dialog typography](~/includes/design/dialogs-image-typography.html)]
